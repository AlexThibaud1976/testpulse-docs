Ce parcours guidé vous mène d'un plan de test à un rapport partageable — sans aucune configuration.

<Steps>
  <Step title="Ouvrir TestPulse" icon="rocket">
    Dans votre projet, allez dans **Test Plans** et ouvrez le hub **TestPulse**. Si vous ne le voyez pas, vérifiez que l'extension est installée (voir [Installation](/fr/getting-started/installation)).
  </Step>
  <Step title="Sélectionner un plan de test" icon="list">
    Choisissez le plan de test à rapporter dans le sélecteur. TestPulse charge ses suites, cas de test et derniers résultats — en lecture seule.
    <Note>Rien n'est encore généré : TestPulse ne fait que lire. Vous pouvez explorer avant d'exporter.</Note>
  </Step>
  <Step title="Vérifier les valeurs par défaut" icon="eye">
    TestPulse pré-remplit un rapport pertinent : titre, page de garde, hiérarchie suites/cas, résultats et traçabilité. Pour un premier essai, acceptez les valeurs par défaut.
  </Step>
  <Step title="Choisir un format d'export" icon="file-export">
    Sélectionnez un format :
    <CardGroup cols={2}>
      <Card title="PDF" icon="file-type-pdf">Soigné, imprimable, signé.</Card>
      <Card title="Word" icon="file-type-doc">Éditable, signé.</Card>
      <Card title="Excel" icon="file-type-xls">Orienté données.</Card>
      <Card title="HTML / PowerPoint" icon="presentation">Partage web / slides.</Card>
    </CardGroup>
  </Step>
  <Step title="Générer" icon="bolt">
    Cliquez sur **Générer**. TestPulse construit le document **dans votre navigateur** et le télécharge. Voilà votre premier rapport.
  </Step>
</Steps>

## Ce que vous venez de produire

Par défaut, votre rapport inclut la **hiérarchie suites → cas** du plan, le **dernier résultat** par cas, la **traçabilité** vers les exigences et une **page de garde**. Tout cela s'affine ensuite.
