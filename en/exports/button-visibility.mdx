---
title: "Show or hide export buttons"
description: "Choose which export buttons appear in the toolbar — saved per user."
icon: "eye"
---

In **Settings → Visible export buttons**, show or hide each export button in the report toolbar.

- Hiding **PDF** also disables the **PDF section order** panel.
- **PowerPoint** can be enabled and generates a 5-slide deck (Cover, KPIs, Results, Traceability, Bugs).
- Settings persist **per user**, via Azure DevOps Extension Data.

<Note>Hiding a button only affects the toolbar — it never changes the content of a generated report.</Note>
